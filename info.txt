¡Buena pregunta!
(9 <= 10) es una comparación. Su resultado es:

VERDADERO (o 1) si la condición se cumple.

FALSO (o 0) si no se cumple.

En tu ejemplo:

(9 <= 10) → VERDADERO → numéricamente 1.

(11 <= 10) → FALSO → numéricamente 0.

(10 <= 10) → VERDADERO → 1.

Por eso nuestra fórmula binaria funciona sin IF:

(DIA_DEL_PERIODO <= 10) * 100


Si el día es 9: (9 <= 10) = 1 → 1 * 100 = 100%.

Si el día es 10: (10 <= 10) = 1 → 1 * 100 = 100%.

Si el día es 11: (11 <= 10) = 0 → 0 * 100 = 0%.

Nota práctica: en Excel verías VERDADERO/FALSO; en tu motor esto se interpreta como 1/0 para poder multiplicar. Si algún día vieras que devuelve texto y no deja multiplicar, usamos un “cast” seguro: ((DIA_DEL_PERIODO <= 10) * 1) * 100.
